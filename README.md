# Enhanced

A Client mod for miniblox to enchance it visually

Features:
- Simple Shadows
- Better Ambient Lighting
- Freecam (Toggle by f4)
- Switch Main Hand (Left/Right)
- Camera Overhaul
- Also with CrazyGames Support
- And many more

See your world from the view like never before.
